Before you begin

NXLV Shield v2 — Lovable Portfolio Audit

What this scanner does:

  • Lists projects from your authenticated Lovable account
  • Probes 4 Lovable API endpoints to measure BOLA/IDOR exposure
  • Downloads files only if you grant explicit deep-inspect permission
  • Analyzes patterns locally — nothing leaves your browser
  • Stores only hashes and masked values — never raw secrets

What this scanner does NOT do:

  • Does not access projects of other users
  • Does not send data to external servers
  • Does not persist file content — only SHA-256 hashes
  • Does not make write requests (POST/PUT/DELETE)

Your responsibility:

This tool scans only projects you own. Using it to access third-party projects is illegal and violates Lovable's Terms of Service.
